Правила использования:

Для получения стикера ячейки "Осн" следует ввести букву "m" перед адресом самой ячейки.

Пример ввода для получения ячейки "Осн" "Осн 2-1-1": m211;

Для получения стикера ячейки "Ящ." следует ввести букву "b" перед адресом самой ячейки.

Пример ввода для получения ячейки "Ящ." "Ящ. 1-1-1": b111;

Для получения стикера ячейки "Зал" следует ввести букву "h" перед адресом самой ячейки.

Пример ввода для получения ячейки "Зал" "Зал 1-1-2": h112;

Примечание: при открытии окна печати, в настройках печати следует выбрать только первую странницу, а также установить маштаба на "166"!

Предупреждение!

В данный момент осутствует информация по таким ячейкам:

  • Все ячейки основного стелажа №1 ("Осн 1"), такие как: "Осн 1-1-1", "Осн 1-1-2", "Осн 1-1-3" и т.д.
  • Ячейка "Ящ. 1-1-17", "Ящ. 1-1-19", "Ящ. 1-1-20" и дальше по нарастающей...
  • Ячейка "Зал 1-1-1"

Следовательно создать для них стикер невозможно!!!

Пример хорошего ввода:

  • Хотим получить стикер с адоресом "Осн 2-2-1", вводим: m221;
  • Хотим получить стикер с адоресом "Ящ. 1-1-9", вводим: b119;
  • Хотим получить стикер с адоресом "Зал 1-1-3", вводим: h113;
  • Хотим получить стикер с адоресом "Осн 9-4-1", вводим: m941;

Пример плохого ввода:

  • Хотим получить стикер с адоресом "Осн 2-1-1", вводим: 211;
  • Хотим получить стикер с адоресом "Осн 2-1-1", вводим: 2-1-1;
  • Хотим получить стикер с адоресом "Ящ. 1-1-1", вводим: 111;
  • Хотим получить стикер с адоресом "Ящ. 1-1-1", вводим: b1-1-1;
